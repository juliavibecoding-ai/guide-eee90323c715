AI-Контент-стратег

AI-сотрудники

Готовый промт

Выступи в роли консультанта по внедрению AI-сотрудников в малый бизнес.

Моя задача: настроить роль AI-Контент-стратег как AI-сотрудника для ежедневной работы бизнеса.
Мне нужен конкретный результат, который можно сразу использовать в бизнесе, контенте, продажах или переписке.

Контекст:
Мой бизнес: [описание бизнеса].
Моя аудитория: [кто эти люди, их ситуация, боли, желания].
Мой продукт/услуга: [описание].
Какие задачи хочу делегировать AI: [список].
Какой формат результата мне удобен: [таблица / список / план / текст].

Если данных недостаточно, сначала задай до 5 уточняющих вопросов. Не переходи к финальному ответу, пока не станет понятно, что именно нужно сделать.

После этого дай результат в структуре:
1. Роль AI-сотрудника и зона ответственности.
2. Какие данные ему нужны для работы.
3. Первые задачи, которые можно делегировать.
4. Формат ответа, удобный для внедрения.
5. Правила качества и ограничения.

Пиши простым, уверенным и профессиональным языком. Избегай общих фраз, клише и пустых обещаний. Делай акцент на смысле, конкретике, пользе для клиента и понятном следующем шаге.

После копирования промт можно сразу вставить в ChatGPT.
